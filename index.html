2D Projectile Motion Simulator

Welcome to the 2D Projectile Motion Simulator! In this sandbox environment, you will be able to manipulate takeoff angle, initial speed, and gravity. A visual simulation will run to demonstrate your system in action!

Extra information:

  • Grid units are in meters.
  • Default gravitational acceleration constant used is 9.80665m/s2.
  • Initial speed is in meters per second.
  • Launch angle is in degrees.
  • The ball may fly out of the visual field, however, it is still in motion so the simulation will not stop until it lands.
  • MASTER RESET IS TO BE USED TO RESET FIELD WHILE BALL IS IN MOTION.
